# AI Journey Portfolio

A premium, interactive developer portfolio showcasing AI-assisted development projects. Built with Next.js, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework**: Next.js 16 (App Router, TypeScript)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Analytics**: Vercel Analytics + Speed Insights
- **Deployment**: Vercel

## Features

- Dark mode, hacker-aesthetic design
- Animated hero section with smooth text fade-in
- Bento grid project layout with hover interactions
- Terminal-styled About section
- Responsive across all breakpoints
- Vercel Analytics integrated from day one

## Projects Featured

### Gulf India Crisis Resource Hub
Orchestrated a fully autonomous agentic workflow to build, deploy, and self-update an emergency resource hub for Indian expats during the 2026 Gulf Crisis — in under an hour.
